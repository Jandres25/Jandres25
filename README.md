## Sobre mi

Ingeniero de Sistemas graduado desde Santa Cruz de la Sierra, Bolivia. Me especializo en desarrollo backend con PHP y arquitecturas MVC, construyendo soluciones web para negocios reales.

Fundador de [@WorkTeam01](https://github.com/WorkTeam01), donde desarrollo sistemas comerciales: gestión de ventas, inventarios, reservas hospitalarias y dashboards administrativos.

- Desarrollo backend con **PHP**, **Laravel** y patrones **MVC**
- Diseño de bases de datos con **MySQL** y **MariaDB**
- Interfaces administrativas con **AdminLTE** y **Bootstrap**
- Experiencia en proyectos comerciales para clientes reales
- Expandiendo conocimientos en **Python**, **Django** y **APIs REST**

## Proyectos Destacados

### [php-mvc-admin-starter](https://github.com/Jandres25/php-mvc-admin-starter)
Starter template con arquitectura MVC personalizada, autenticación, gestión de usuarios y permisos. Construido con PHP 8.2+, AdminLTE 3 y DataTables.

`PHP` `MVC` `AdminLTE` `Bootstrap` `DataTables` `TCPDF` `Authentication`

### [Sistema de Ventas PHP](https://github.com/WorkTeam01/Sistema_de_Ventas_PHP)
Sistema completo de ventas con gestión de inventario, facturación PDF y dashboard administrativo. Desarrollado para uso comercial real.

`PHP` `MySQL` `AdminLTE` `Bootstrap` `TCPDF` `Inventory`

### [Sistema de Reservas Hospitalarias](https://github.com/WorkTeam01/SistemaReservasHospital)
Sistema de gestión de citas medicas para clinicas y hospitales. Incluye wizard de reservas de 5 pasos, dashboards por rol (admin, doctor, recepcionista), reportes exportables y calendario de citas.

`PHP 8.2` `MVC` `MySQL` `AdminLTE 3` `Chart.js` `DataTables` `jQuery`

### [Conversor de Monedas](https://github.com/Jandres25/ChallengeConversorMonedaJava)
Conversor de monedas en Java que consume la API de ExchangeRate para tasas de cambio en tiempo real.

`Java` `API REST` `ExchangeRate API`
